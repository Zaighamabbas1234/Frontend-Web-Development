The Pros and Cons of using Jupyter Notebook:-
Pros:
•	All in one place.
•	Easy to convert.
•	Easy to share.
•	Language independent.
•	Interactive coding.
Cons:
•	It is almost impossible to practice good code versioning.
•	The non-linear workflow of jupyter.
•	Jupyter is bad for running long asynchronous tasks.
•	Jupyter can be slow.
•	No IDE integration.
